# lid_opm
Exploring cortical patterns of levodopa induced dyskinesia using OPM-MEG.


# Dependencies
`conda create -n lid_opm python numpy pandas scipy scikit-learn jupyter matplotlib`
additional packages:
`pip install mne h5io pymatreader`

package version:
- Python sys 3.12.7
- numpy 1.26.4
- mne 1.8.0
